### API

Interfaz de programación de aplicaciones.

Interfaz: programa común para que 2 sistemas se puedan comunicar entre si.

ej:google brinda una interfaz programada en JavaScript que por debajo se conecta a los sistemas de Google Maps.

### REST

Representational State Transfer (Transferencia de Estado Representacional)

Es un estilo de arquitectura que define un conjunto de principios para que lo usuarios se comuniquen.
